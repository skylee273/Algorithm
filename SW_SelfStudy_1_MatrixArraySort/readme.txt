[선택정렬]
선택정렬(Selection Sort)이란? 주어진 자료들 중 가장 작은 값의 원소부터 차례대로 선택하여
위치를 교환하는 방식이다. 선택 정렬은 셀렉션 알고리즘을 전체 자료에 적용한 것이다. 

저장되어 있는 자료로부터 k번째로 큰 원소 혹은 작은 원소를 찾는 방법이다. 최소값, 최대값 혹은 중간ㄱ밧을 찾는 알고리즘을 
의미하기도 한다. 셀렉션 알고리즘의 선택과정은 다음과 같다. 

[ 선택과정 ]
* 정렬 알고리즘을 이용하여 자료 정렬하기
* 원하는 순서에 있는 원소 가져오기

[ 알고리즘 ]
일반적인 셀렉션 알고리즘은 다음과 같다. 아래는 k번째로 작은 원소를 찾는
알고리즘 과정을 나타낸다. 

* 1번부터 k번쨰까지 작은 원소들을 찾아 배열의 앞쪽으로 이동시키고, 배열의 k번째를 반환한다. 
* k가 비교적 작을 때 유용하며 O(kn)의 수행시간을 필요로 한다.
