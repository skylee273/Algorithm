입력으로 1개의 정수 N 이 주어진다.

정수 N 의 약수를 오름차순으로 출력하는 프로그램을 작성하라.
 

[제약사항]

N은 1이상 1,000이하의 정수이다. (1 ≤ N ≤ 1,000)
 

[입력]

입력으로 정수 N 이 주어진다.


[출력]

정수 N 의 모든 약수를 오름차순으로 출력한다.

[풀이]
단순히 나머지가 0인 방법으로 풀수도 있는 문제이다. 1~N 까지 하지만 N이 큰 수라고 가정한다면 수행시간은 N이 될것이다. 
이것도 현대 컴퓨터에서는 충분히 구할 수 있으나 조금 더 줄 일 수 있는 방법은 약수는 항상 쌍으로 존재하는 것을 이용하여
수행시간을 절반으로 줄일 수 있다.
