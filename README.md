# Algorithm
알고리즘 관련  공부 내용과 소스 파일

# 단축키 모음
CTRL - K,C 선택한 블럭을 전부 코멘트  
CTRL - K,U 선택한 블럭을 전부 언 코멘트  
Ctrl-F7 : 현 파일만 컴파일  
Ctrl + Shift + A 파일 추가  
Ctrl + i 증분 찾기 ( 오류 디버깅 에서 찾을 때 좋음 )  
Ctrl + Shift + S 모두 저장  
Alt + Enter : 속성  
Ctrl + ; 솔루션 탐색기 이동  
Ctrl + J : 자동 완성   
F9 : 중단점 생성
Ctrl + Shift + F6 : 문서 창내 탭간 이동  
Ctrl + Shift + Tab : 열려진 문서들 간 선택 이동   
Ctrl + F4 : 문서창내 탭 닫기
