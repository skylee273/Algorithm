## 99. Shortcut
* VisualStudio 2015 C++
    * [ CTRL + K + C ] - 선택한 블럭을 전부 코멘트
    * [ CTRL + K + U ] - 선택한 블럭을 전부 언 코멘트 
    * [ CTRL + F7] - 현 파일만 컴파일
    * [ CTRL + Shift + A ] - 파일 추가
    * [ CTRL + i ] - 증분 찾기 ( 오류 디버깅 에서 찾을 때 좋음 )
    * [ CTRL + Shift + S ] - 모두 저장
    * [ Alt + Enter ] - 속성
    * [ CTRL + ; ] - 솔루션 탐색기 이동
    * [ CTRL + j ] - 자동 완성
